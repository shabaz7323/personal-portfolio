# Personal Portfolio - Shabaz Ahamed

## Quick start
1. npm install
2. npm run dev
3. Open http://localhost:5173

## Notes
- This project uses React + Vite + Tailwind (devDeps included in package.json).
- Replace `public/resume.pdf` with your actual resume.
- Replace images in `public/images/` with your own photos.
- Update the contact form email in `src/components/Contact.jsx` to your real email.
- Update social links in `src/components/Footer.jsx` to your real profiles.

## Deploy to Vercel
1. Push this repo to GitHub.
2. Import the repo into Vercel (detects Vite automatically) and deploy.
